# TreeKeeper – Data Entry Instructions

> Quick guide to view sites needing updates, update existing trees, add new trees, and move sites.

## To View Sites That Need Updating

1. Log onto **TreeKeeper** *(arrow coming out of box at top of map)*.
2. Select **Search** *(magnifying glass on left panel)*.
3. Select **Quick Filters**.
4. Choose **UFA Trees to Update** *(towards bottom of drop-down list)*.
5. Sites/trees that need updating will appear **highlighted in yellow**.

## Updating an Existing Site (tree)

1. Log onto **TreeKeeper**.
2. Select a tree by tapping a **circle (tree)** on the screen.
	 *Note: It may take multiple tries for the app to recognize finger inputs.*
3. Open the selected tree by clicking **View Site**.
4. Press the **Edit** button *(upper-right — green box with pen)*.
5. Click **Species**.
6. **Update** the data.
7. Press **Submit**.

## Adding a New Site (tree)

1. Log onto **TreeKeeper**.
2. Press your finger on the desired new tree location. A **red raindrop with a black star** will appear.
3. Select **Tools**.
4. Select **Add Point** *(bottom of screen)*.
5. Click the **menu bar** *(three horizontal lines)*.
6. Click the **red raindrop**.
7. Select **Load Last**.
	 *Must have previously viewed an adjacent tree for “Load Last” to work.*
8. Enter data and click **Submit**.
9. If the tree icon is not exactly where you want it, use the **Move Site** process to place it precisely.
10. **Unlimited moves** are possible after the initial placement.

## Moving a Site

1. Select the tree that you want to move.
2. In the **Site Information** box, select **Move Site** *(located at the bottom of the box)*.
3. A **blue raindrop** will appear.
4. Press your finger on the blue icon and drag it to the desired location.
	 The **new location** will be **blue** and the **old location** will be **orange**.
5. If satisfied with the new location, click the **check mark** *(top right of the screen)*.
	 If not satisfied, click the **red X** and repeat the process.

## Changing a Tree to Vacant Site or Stump

1. Select a **tree**.
2. In the **Site Information** box, choose **Apply Rule / Enter Value** *(dropdown)*.
3. Choose **“Change to Stump Rule”** or **“Stump to Vacant Site.”**
4. Select **Apply Rule**.

## Deleting a Site

1. Log onto **TreeKeeper**.
2. Select the target tree by touching the screen to **activate** the site.
3. Open **Site Information** by clicking **View Site**.
4. Select the **Edit** icon.
5. Select **Delete** at the bottom of the screen.
	 *It may take a few minutes for the tree to be removed from the map — a refresh removes it immediately.*

## See the Sites You Personally Updated

1. Log onto **TreeKeeper**.
2. Select **Menu**.
3. Select **Full Version** *(bottom of screen; TreeKeeper may refresh)*.
4. Select the **Search** icon *(left side menu)*.
5. Select **Filters** → **Site**.
6. Scroll to the bottom to **Last Changed By**.
7. Enter your name into **Last Changed By**.
8. Select **Submit**.
	 *All trees you inventoried will appear as **yellow dots**.*

## Follow‑Up Notes

- During data entry, you may be asked **“Proceed with sequencing?”**
	**No** for an existing site; **Yes** for a new site. *(Feature may have changed in the latest TreeKeeper update.)*
- If a command you expect doesn’t appear, make sure you’ve **scrolled to the bottom** of the information box.
- If finger taps are unreliable on the map, consider using a **pen stylus**.
