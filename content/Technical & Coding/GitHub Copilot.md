## GitHub Universe 2023 Keynote

- Youtube video:
	- [GitHub Universe 2023 Opening Keynote](https://www.youtube.com/watch?v=NrQkdDVupQE&t=25s) ✅ 2023-11-11
	- **Introduction to AI and Software Development**
		- GitHub CEO Thomas Dohmke discussed the promise of AI in software development. He highlighted how AI can help developers write code faster, improve code quality, and automate repetitive tasks.
	- **Overview of GitHub Copilot Chat**
		- Allison Weins provided an overview of GitHub Copilot Chat, a new feature that allows developers to interact with GitHub Copilot using natural language. This feature can be used to generate code, explain code, and more.
	- **Build an App with GitHub Copilot Chat**
		- A demonstration was given on how to build an app with GitHub Copilot Chat. The demonstration showed how to use GitHub Copilot Chat to generate code, write tests, and deploy the app to GitHub.
	- **GitHub Copilot Chat in Visual Studio, Visual Studio Code, JetBrains IDEs**
		- GitHub Copilot Chat is now available in Visual Studio, Visual Studio Code, and JetBrains IDEs. This makes it even easier for developers to use GitHub Copilot Chat on their favorite IDEs.
	- **GitHub Copilot Chat in GitHub Mobile**
		- GitHub Copilot Chat is also now available in GitHub Mobile. This means that developers can now use GitHub Copilot Chat on their mobile devices.
	- **GitHub Copilot Chat Gets Even More Powerful**
		- GitHub Copilot Chat is constantly being improved. New features were announced, including the ability to generate code from natural language descriptions, write code comments, and more.
	- **GitHub Copilot across the Developer Workflow**
		- Kedasha Kerr discussed how GitHub Copilot can be used across the entire developer workflow. She showed how GitHub Copilot can be used to write code, test code, and deploy code.
	- **GitHub Copilot Chat and Unit Tests**
		- GitHub Copilot Chat can now be used to write unit tests. This can help developers to write higher quality code.
	- **GitHub Copilot Chat in GitHub.com**
		- GitHub Copilot Chat is now available in GitHub.com. This means that developers can now use GitHub Copilot Chat to explain code, generate pull request descriptions, and more.
	- **GitHub Copilot Chat in GitHub Advanced Security**
		- GitHub Copilot Chat is now integrated with GitHub Advanced Security. This means that GitHub Copilot Chat can now be used to autofix code scanning issues and identify generic secrets.
	- **GitHub Copilot Personalized to an Organization's Codebase**
		- Colin Merkel discussed how GitHub Copilot can be personalized to an organization's codebase. This can help developers to write code that is consistent with the organization's coding style.
	- **Fostering an Ecosystem Built on Copilot**
		- GitHub announced the GitHub Copilot Partner Program, which will help to foster an ecosystem of tools and services that work with GitHub Copilot.
	- **Accenture + GitHub Copilot**
		- Accenture announced that they are partnering with GitHub to bring GitHub Copilot to their clients.
	- **GitHub Copilot Enterprise**
		- GitHub announced GitHub Copilot Enterprise, a new offering that is designed for large organizations.
	- **Microsoft CEO Satya Nadella + GitHub CEO Thomas Dohmke**
		- Microsoft CEO Satya Nadella and GitHub CEO Thomas Dohmke discussed the future of software development. They highlighted the importance of AI in software development and the role that GitHub Copilot will play in the future of the industry.
	- **Closing Remarks**
		- The event concluded with closing remarks from GitHub CEO Thomas Dohmke. He thanked the attendees for their participation and announced that GitHub Universe will be back in 2024.
	- **GitHub Workspace**
		- GitHub announced GitHub Workspace, a new tool that will help developers to manage their code, collaborate with others, and deploy their code.
