# My Thoughts on Code Bootcamps

These were my thoughts as shared to a sibling of a close friend
- Date: #🗓️/2023/04/10

```
Hi Friend

Whether to take on $20,000 of debt to attend a coding boot camp is a good question to consider.

It's quite possible that it will pay off, but there's no guarantee. Especially regarding timing (how soon you may get the job)

If you decide to take a loan, I recommend finding one that only requires repayment when you're employed if at all possible.

This was the case for the one my brother attended (Lambda School, now named Bloomtech(https://www.bloomtech.com/)). I think they call it their 110% Tuition Guarantee. I see that [Galvanize / Hack Reactor have a similar offer](https://www.galvanize.com/hack-reactor/cost-financing/). It appears to be known as an ISA (Income Share Agreement). His loan was based on a percentage of his salary, which was convenient, and it was only due if he had a job. Some boot camps may offer similar options, so it's worth looking into. I didn't see this option with Flatirons, but you may be able to dig further. The loan is completely worth it if you have a job in tech, but less so if you don't… and even less so if no job at all.

As to the course work of Flatirons, I downloaded the syllabus and it looked sufficient. All good subjects, and it appeared rigorous enough to be useful.  
  
Before starting the boot camp, I'd highly recommend learning the basics of programming. There are several resources available, such as freeCodeCamp or Code Academy, which can help you get started. If you don't know where to begin, you can look at the Flatirons syllabus / curriculum to familiarize yourself with the topics you'll be studying. 

Having this background knowledge will make it easier to understand the boot camp's material and increase your learning speed.
```
